Ubuntu Repository

Repository

An Ubuntu PPA repository is provided for VLC-Qt for all supported Ubuntu releases.

$ add-apt-repository ppa:ntadej/tano
Warning: This PPA may contain some backports or patches of various applications and libraries that are required for VLC-Qt and/or other projects in this repository.

Clean archive repository

$ add-apt-repository ppa:ntadej/tano-clean

Installation

Install VLC-Qt using your package manager or using command line:

$ apt-get install libvlc-qt-core1 libvlc-qt-widgets1 libvlc-qt-dbg libvlc-qt-dev
Warning: You might need administrative privileges for configuration and installation.

Repository information

This repository is signed with 1024R/55D8172B OpenPGP key. Follow these instructions if you experience problems installing packages from this PPA or use this command to add OpenPGP key:

$ apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 55D8172B
